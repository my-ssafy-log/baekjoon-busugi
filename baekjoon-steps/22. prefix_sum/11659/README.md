# 11659 구간 합 구하기 4

[링크](https://www.acmicpc.net/problem/11659)

## 문제 해설
이 문제는 나열된 N개의 수들 중 M개 입력되는 i, j값에 대해서 수열 i번 부터 j번까지의 합을 구하는 구간합 문제이다.

이 문제는 누적합 배열을 만들어 구간합을 구하는 풀이를 진행했는데, 10만번의 연산에도 불구하고 시간 초과가 떴다.
